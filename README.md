# ircbloq-resource

![](https://img.shields.io/travis/com/ircbloqcc/ircbloq-resource) ![](https://img.shields.io/github/license/ircbloqcc/ircbloq-resource)

Provide a local resource server for ircbloq.

### Instructions

```bash
npm install
npm run fetch
npm start
```

To test upgrade funciton, run:

```bash
npm test
```
